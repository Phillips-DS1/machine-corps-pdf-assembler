# Quick Start Guide â€“ For Non-Technical Users

1. **Launch the Tool**:
   - Double-click `launch.bat` (Windows) or `launch.sh` (macOS/Linux).
   
2. **Set Input/Output** (Main Controls tab):
   - Browse for Markdown folder (e.g., inputs/).
   - Set output PDF path.
   - Add classification (e.g., CUI // FOUO) for banners.
   
3. **Edit Cover** (Cover Editor tab):
   - Modify text lines.
   - Adjust styling (font, size, bold).
   - Choose BG color.
   - Slide padding.
   - Click "Update CSS & Refresh Preview".
   - "Save as Default" to persist.

4. **Edit Header/Footer** (Header/Footer tab):
   - Edit text (use {page} for numbering).
   - Style and height.
   - Update color.
   - "Update CSS & Refresh Preview".
   - "Save as Default".

5. **Review Preview** (Live Preview tab):
   - Click "Refresh Preview" to see WYSIWYG.
   - Export HTML for sharing.

6. **Build PDF**:
   - Click "BUILD PDF" at bottom.
   - Open generated file when prompted.

Tips: Changes bind to config.yaml on save. For issues, check status bar.
